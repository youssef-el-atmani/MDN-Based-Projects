The Only Way To Go Fast Is To Go Well

I created this project system to solve real problems I faced in the past while learning web development.

I follow something I call Cumulative Learning. It’s similar to the original concept, but with my own twist:
Every time I learn something new, I build a new project that includes the new skills plus everything I’ve learned before .
