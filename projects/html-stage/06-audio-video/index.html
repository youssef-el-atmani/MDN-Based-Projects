How to Embed Audio & Video

The new covered lesson is about how to embed audio and video elements into a webpage
for video we use the video element, and for audio we use the audio element,
To embed them correctly, we have multiple attribute that goes insie the video & audio opening tags,
they accepts almost all the same attributes, except of some attributes, which are:
width, height, poster
and it is obvious why the audio element does not support those attributes.
the rest common attributes are:

  • controls: which give the user some interface to control the video and audio
  • loop: which makes the audio or the video keep replaying when its finish
  • muted: which start the audio or the video muted by default
  • src: to place the location of the media file

We have another way to add the source of the file, using the source element
it is better than the basic src attribute, because we can define multiple formats,
while we can't do that that with the src attribute,
and also we can specify the type of the linked media-file using the type attribute.
That type attr inform the browser of the format of the file it hold,
so that the browser can now if it can play that type of media or not before even to media-file.

How to Add tracks to Videos

I also learned how can I add captions and subtitles to a video
it is done using the track element,
as MDN informed, the track element should be placed under the sources elements.

It accept multiple attributes:

  • src: the location of the track file
  • srclang: The language of which the track is written
  • kind: is it a caption, subtitles, or what?
  • label: Give a label for the language of the track.
  • default: which makes that track plays automatically with the video
